# 📊 Bellabeat Fitbit Data Analysis

## 🔍 Project Overview

This project analyzes Fitbit smart device data to understand user activity patterns, sleep behavior, and sedentary lifestyle trends.
The goal is to generate actionable insights that can help Bellabeat improve user engagement and wellness recommendations.

---

## 🎯 Objectives

* Analyze daily activity and sleep patterns
* Identify relationships between steps, calories, and sleep
* Understand sedentary behavior trends
* Build an interactive dashboard for business insights

---

## 🛠️ Tools & Technologies

* **Excel** → Data cleaning & preprocessing
* **SQL** → Data transformation & analysis
* **Tableau** → Data visualization & dashboard creation

---

## 🔄 Project Workflow

### 1️⃣ Data Collection

* Multiple Fitbit datasets (daily activity, sleep, hourly steps, calories, etc.)
* Raw datasets stored in `data/raw_data`

---

### 2️⃣ Data Cleaning (Excel)

* Removed duplicates and null values
* Standardized column names
* Converted date formats
* Created derived columns (e.g., Sleep Quality categories)

---

### 3️⃣ Data Analysis (SQL)

* Aggregated key metrics (steps, calories, sleep)
* Joined multiple datasets
* Created summary tables for visualization
* Identified trends and relationships

---

### 4️⃣ Data Visualization (Tableau)

* Built an **interactive dashboard** with:

  * KPI Metrics (Avg Steps, Sleep Hours, Calories, Sedentary %)
  * Steps by Day (Bar Chart)
  * Sleep Analysis (Bar Chart)
  * Steps vs Calories (Scatter Plot)
  * Steps vs Sleep (Scatter Plot)
  * Hourly Activity Heatmap

---

## 📊 Key Insights

* Users tend to be more active on weekends
* Positive correlation between steps and calories burned
* Better sleep is generally associated with higher activity levels
* Sedentary time remains consistently high across the week
